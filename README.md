android_device_kttech_e100
==========================

CyanogenMod10 device settings for KM-E100 (Take LTE) - Jellybean Branch

Build requirements:
* device/kttech/msm8960-common, branch jellybean

Working:
* RIL
* Graphics
* Sound
* Wifi
* Tethering
* Bluetooth
* Sensors
* GPS
* Camera
* Video

Known issues:
* HDMI is not tested
* Properiatary softwares are not working ( DMB, SRS, etc...)
